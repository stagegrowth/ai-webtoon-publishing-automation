# As-Is 요약 (고객 확인용) — 업로드 대상 업체별

**업로드 대상 업체(카카오, 네이버, 레진 등)마다 절차가 다르므로, 업체별로 나눠 정리했습니다.**  
각 업체 표에서 "현재 방식(요약)"을 실제로 하시는 절차에 맞게 채워 주시고, 체크리스트를 확인해 주세요. 개발 용어는 몰라도 되며, "실제로 어떻게 하고 계신지"만 알려 주시면 됩니다.

---

## 고객 확인 체크리스트 (업체별)

- [ ] **업체 1** Flow(아래 표) 단계명/현재 방식이 실제와 일치
- [ ] **업체 2** Flow(아래 표) 단계명/현재 방식이 실제와 일치
- [ ] **업체 3** Flow(아래 표) 단계명/현재 방식이 실제와 일치
- [ ] **예외표**의 발생 Step/현재 처리/자동화 희망이 정확
- [ ] **자동화 경계**(완전 자동/반자동/제외) 업체별 배치가 타당
- [ ] 자동화 입력 형식(Excel·파일) 업체별 차이 요구사항 반영

*(업체 1·2·3은 실제 업로드 대상명(예: 카카오, 네이버, 레진)으로 Notion에서 제목을 바꿔 사용하시면 됩니다.)*
