# As-Is 요약 (업로드 서비스 회사별 분리) — Notion 붙여넣기용

이 문서는 As-Is를 **회사별로 완전히 분리**해 정리한 초안입니다. 로그인·권한·검증을 포함한 전 단계가 회사마다 다를 수 있다는 전제를 반영합니다.  
회사 식별자: **COMPANY-A**, **COMPANY-B**, **COMPANY-C** (실제 서비스명으로 치환 가능)

- [ ] **4.** 회사별 **업로드 API 제공 여부** (있음/없음, 문서 링크) — ASIS-x-11 구현 방식
